## Aritri Cha — Multi-Container App on AWS

A production-style full-stack application built with **React, Node.js, MongoDB, and Nginx**, deployed on **AWS** using **Docker** and **Terraform**, with an automated **CI/CD pipeline**.

### 🧩 System + Stack

- **Frontend** → React  
- **Backend** → Node.js (Express)  
- **Database** → MongoDB  
- **Reverse Proxy** → Nginx  
- **Containers** → Docker / Docker Compose  
- **Cloud** → AWS  
- **IaC** → Terraform  
- **CI/CD** → Build → Push → Deploy
