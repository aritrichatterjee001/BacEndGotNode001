## Aritri Cha — Build & Deploy Multi-Container Applications to AWS

A containerized full-stack system built with **React, Node.js, MongoDB, and Nginx**, deployed on **AWS** using **Docker + Terraform**, with a fully automated **CI/CD pipeline**.

---

### ⚙️ Features

- **Multi-container architecture**
  - Frontend → React  
  - Backend → Node.js (Express)  
  - Database → MongoDB  
  - Reverse Proxy → Nginx  

- **Cloud deployment**
  - Scalable AWS infrastructure  

- **CI/CD pipeline**
  - Automated Docker builds  
  - Push to DockerHub  
  - Continuous deployment  

- **Infrastructure as Code**
  - Terraform-managed resources

- AWS  
- Terraform  
- CI/CD (GitHub Actions or similar)

---

### 🔄 Deployment Flow

1. Push code to repository  
2. CI/CD pipeline triggers  
3. Docker images built & pushed to DockerHub  
4. Terraform provisions/updates AWS infrastructure  
5. Containers deployed and served via Nginx
